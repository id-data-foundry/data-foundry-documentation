# Instance Overview

The **Instance Overview** is the landing page of the Admin Portal (`/admin`). It provides a high-level snapshot of the system's health and activity.

## Key Metrics

*   **Total Users:** The number of registered users on the platform.
*   **Active Projects:** The number of currently active projects.
*   **Storage Usage:** An overview of disk space used by uploads and datasets.
*   **System Status:** Indicators for database connectivity and external service (OOCSI, Mail) status.

## Quick Actions

From the overview page, you can typically navigate to:
*   **User Management:** To approve accounts or manage roles.
*   **Project Management:** To review or archive projects.
*   **System Logs:** To check for recent errors or warnings.

Regularly monitoring this dashboard helps ensure the smooth operation of your Data Foundry instance.
