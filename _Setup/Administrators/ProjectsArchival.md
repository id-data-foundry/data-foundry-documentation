# Project Archival

As an administrator, you may need to manage the lifecycle of projects on the platform.

## Archiving a Project

Archiving a project is a way to make it read-only and remove it from the active project lists without permanently deleting the data.

1.  Navigate to the **Project Management** section of the Admin Portal.
2.  Search for the project by name or owner.
3.  Click the **Archive** button next to the project.

## Effects of Archiving

*   **Read-Only:** No new data can be added to datasets in the project.
*   **Visibility:** The project is hidden from the public "Explore" page but remains accessible to the project owner and administrators via their archives.
*   **Restoration:** Administrators can un-archive a project if needed, restoring full functionality.

This feature is useful for cleaning up old student projects at the end of a semester while preserving the work for future reference or grading.
