# Token Usage & Energy Estimation

This example shows how to use the `textToTextWithUsage` function to retrieve token usage statistics and estimate energy consumption.

[View Example]({{ site.baseurl }}/_Guides/LocalAI/Examples/example13.html){: .btn .btn-primary }

## Code Explanation

Instead of using `foundry.textToText`, which returns a simple string, we use `foundry.textToTextWithUsage`. This function returns an object containing both the generated `text` and the `usage` statistics.

```javascript
const response = await foundry.textToTextWithUsage({
  api_token: "YOUR_API_KEY",
  prompt: "Explain the concept of entropy in simple terms.",
});

if (response) {
  console.log("Text:", response.text);
  console.log("Prompt Tokens:", response.usage.prompt_tokens);
  console.log("Completion Tokens:", response.usage.completion_tokens);
  console.log("Total Tokens:", response.usage.total_tokens);
}
```

By knowing the number of tokens, you can calculate a rough estimate of the energy consumed by the LLM during generation. In the example, we use a constant factor to demonstrate this principle.
