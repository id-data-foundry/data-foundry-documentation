# Prompt Chaining
In this example we will show you how to get started with chaining Text-to-Text prompts. Prompt Chaining will allow you to script out multi step prompts. This can be quite handy for when you have to do complex tasks, as prompt chaining allows you to cut down complex tasks, into bite sized chunks. These bite-sized chunks make tasks easier for AI models and helps you gain more reliable results.
