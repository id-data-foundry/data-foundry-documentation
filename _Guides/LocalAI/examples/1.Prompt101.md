# Text-to-Text
To get started with Data Foundry AI, you should first get started with the most basic example; Working with a text-to-text AI model. This model allows you to generate a text response based on a given prompt. Data Foundry AI provides several text-to-text models to help you generate text from text. In the extra settings you can find the temperature slider, which controls the randomness of the generated text. A higher temperature will result in more random text, while a lower temperature will result in more predictable text.
