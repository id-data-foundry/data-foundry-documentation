# Data-to-Text
In the Data-to-Text example, we show you how you can use 2 separate prompts to modify data. In this example we have one prompt that requests data, and one prompt that allows the user to describe how the data should be modified. This shows you that not all prompts have to be manually created, and that you can still use data as a variable in your prompts.
