### Data Foundry Client Guides

In the Data Foundry Client guides you can learn how to connect to Data Foundry using your programming language or platform of choice. We have guides for a variety of creative coding and data science platforms.

*   **[Python]({% link _Guides/Clients/Python.md %})**: Use Data Foundry with standard Python scripts or notebooks. Ideal for data analysis scripts, Raspberry Pi prototypes, and backend logic. You can send data using `requests` or use the `oocsi` library. You can also download your data.
*   **[JavaScript]({% link _Guides/Clients/Javascript.md %})**: Connect web applications and hosted websites. You can use `fetch` requests to send data directly to the API, or use the OOCSI library for real-time communication in your web-based prototypes and visualizations.
*   **[Processing]({% link _Guides/Clients/Processing.md %})**: Integrate data into your Processing sketches. We provide a library that makes it easy to send data from your creative coding sketches, either directly or through OOCSI.
*   **[Unity]({% link _Guides/Clients/Unity.md %})**: Log data from 3D environments and games. Our guide shows you how to integrate data collection into games and VR/AR experiences using OOCSI.
*   **[Max]({% link _Guides/Clients/Max.md %})**: Connect your interactive media and sound projects. This guide explains how to use `XMLHttpRequest` within Max to send data from your patches to Data Foundry.
