# Add collaborators to a project

To add collaborators to your Data Foundry project, all you have to do is:

1. go to the **project page** of the project you want to add a collaborator. 
2. after which you **press the add collaborator button** underneath the researcher section. 
3. Here you have to **fill in their** institution (University/School/Company) **email**
4. Now you have to **press the green ADD button**. 
5. The collaborator will now receive an email containing an invite to the project which they can then either accept or deny.

# Remove collaborators from a project

Sometimes you might want to remove collaborators from your Data Foundry project, maybe if you accidentally have added the wrong person to the project, or has asked someone to remotely troubleshoot your project. To remove a collaborator from a Data Foundry project:

1. you have to first **open the project** in question
2. then **navigate to the researchers section** underneath the info pane. 
3. In the researcher section you can then **click the tiny trash** can **next to the collaborators name** to remove them from the project.

The collaborator will now be removed from the project, and will not be able to access the data anymore.
