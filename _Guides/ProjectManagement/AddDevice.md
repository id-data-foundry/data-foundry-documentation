# Adding a device to a project

Inside a project that you are the project owner of (you created the project yourself):

1. Click on the **sources tab*** on project page. 
2. Click **add new device** under the tab devices.
3. Fill in the necessary details and click on **save**.
4. Click again on the **device icon**, now you can see its unique ID

{: .info}
This will only be possible for the project owner.
