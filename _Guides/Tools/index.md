# Data Foundry Tools

Beyond file storage, clients and scripting, Data Foundry includes several built-in tools to streamline your design and research workflows. These tools provide locally hosted alternatives to common services, ensuring your data remains private and easily accessible.

## Available Tools

- **[Starboard Notebooks]({% link _Guides/Tools/Starboard/StarboardNotebook.md %})**: Interactive Python-style notebooks for data analysis without the need for local installation.
- **[Transcription]({% link _Guides/Tools/Transcription.md %})**: Automatically generate text from audio recordings.
- **[RawGraphs]({% link _Guides/Tools/Rawgraphs.md %})**: Create professional data visualizations directly from your datasets.
- **[ESP Tools]({% link _Guides/Tools/ESPtools/MicropythonSuite.md %})**: Specialized tools for managing ESP32 and MicroPython devices.
