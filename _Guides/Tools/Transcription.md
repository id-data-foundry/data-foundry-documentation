# Interview Transcription

Data Foundry offers a built-in tool for transcribing audio and video files, powered by OpenAI's Whisper model running locally on our servers. This ensures your data remains private and secure within the Data Foundry environment.

## How to use it

1.  Navigate to the **Tools** section in the main menu.
2.  Select **Transcription**.
optional: change the transcription language and quality in the dropdown menu.
3.  **Upload** your audio (`.mp3`, `.wav`) or video (`.mp4`) file.
    *   *Note: Large files may take some time to process.*
4.  Once processing is complete, the text will appear on the screen.
5.  You can **Edit** the text directly if there are minor errors.
6.  **Download** the transcript as a text file or copy it to your clipboard.

## Supported Languages

The tool supports multiple languages including English and Dutch.
