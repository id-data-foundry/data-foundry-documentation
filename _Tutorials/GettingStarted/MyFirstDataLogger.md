# Tutorial 2 - My First Datalogger!
### Getting started with collecting data with Data Foundry

![Dataflow]({% link _Tutorials/GettingStarted/images/DF_Blocks_T2.jpg %}){:width="20%"}{: style="float: right"}

Now you can get around Data Foundry, we can get started showing you what Data Foundry is capable of. We will do this by turning your phone into a simple Data Foundry datalogger and using this to log data into a IoT Dataset. After which we will look at the generated data and generate a small graph. Once again, we have created a small on tour which will take you through all the steps.

{: .info}

For this tutorial you will need a mobile device like a phone or tablet.
