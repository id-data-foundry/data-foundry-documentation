# Tutorial 4 - My First Data Foundry Website!
{: .no_toc }
### Hosting a simple website on Data Foundry

![Dataflow]({% link _Tutorials/GettingStarted/images/DF_Blocks_T4.jpg %}){:width="20%"}{: style="float: right"}
In the previous tutorial we taught you how to get data from the real world into Data Foundry and make simple visualisations with this data. Now we are going to learn how you can use Data Foundry for prototyping.In this tutorial we will build a simple website and host this on Data Foundry. The purpose of this tutorial therefore is to get acquainted with the tools to get your website/mockup/files up and running.
