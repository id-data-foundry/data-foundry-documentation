# Hi there, welcome to Data Foundry!

Cool that you want to start using Data Foundry. You are probably here as you want to get started with the platform, but do not know where to start. 

Understandably so, because Data Foundry offers so many possibilities, it is easy to get overwhelmed. But reading this text here is not going to help much with that. That's why we made a small tour for you to explore the platform. This way, you can get acquainted with the platform and get a gripe of where everything is.

## Interface Overview

To make it easier for students to onboard onto Data Foundry, we created interactive Data Foundry tours. These tours will take your students through the process of setting up a Data Foundry project, step-by-step in a way that is almost foolproof. This is a set of interactive tutorials that can get you started with the Data Foundry platform, explaining you the basics of data collection and prototyping using the platform. For beginners it is recommended to start at the first tutorial as this will give you a general impression of the platform. These tutorials will guide you through the entire process; from making your first dataset to a fully functional prototype.

This introductory tutorial guides you through the most important menus of the Data Foundry platform, showing you where to find core functionalities.
