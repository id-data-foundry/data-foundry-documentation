# Tutorial 2 - My First Connected Datalogger!
### Getting started with collecting data through an esp32 with Data Foundry

As tutorial 2 already introduced you to the concept of dataloggers, we are stepping this up a notch, by turning a microcontroller into a datalogger. But for this we first need to collect a couple of supplies.

### Shopping List:
- 1 x Waveshare ESP-32s3 pico
- 1 x DHT11 temperature sensor
- 3 x Jumper wires

Any other ESP-32 based boards would also work, however it might be simpler to use this board.
