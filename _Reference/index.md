# Reference

This section contains technical reference documentation. It is organized for quick look-up and provides the specific, factual details needed to work with Data Foundry's APIs, client libraries, and data structures. Come here when you know *what* you need to do and want to find out *how* to do it.

- **[Data Foundry Developer API]({% link _Reference/DFapi.md %})**: This is the starting point for all programmatic interaction. It provides the technical specifications for authenticating with the platform, including how to generate and use the necessary API keys for your project.

- **[Data Foundry Clients]({% link _Reference/Clients/index.md %})**: A collection of technical manuals for the official client libraries. Look here for function signatures, code examples, and class definitions needed to interact with Data Foundry from environments like MicroPython and Processing/Java.

- **[Datasets]({% link _Reference/Datasets/index.md %})**: This section details the technical structure and schema of the various dataset types. Use this as a reference when you need to parse data exports or generate data that conforms to the platform's expected format.

- **[Participant API]({% link _Reference/ParticipantAPI/index.md %})**: The complete technical reference for the client-side JavaScript API. This guide provides the function definitions and code snippets required to build custom web interfaces that allow study participants to interact securely with their own data.

- **[Scripts]({% link _Reference/Scripts/index.md %})**: The definitive guide to the server-side scripting engine. This is a complete reference of all available `DF` functions for automating tasks, transforming data, and triggering events directly on the platform.

- **[Local AI]({% link _Reference/LocalAI/index.md %})**: Your technical guide to the AI functionalities. This section contains the detailed API reference for services like Text-to-Speech and language models, including endpoints, request/response formats, and a look-up list of available models.
